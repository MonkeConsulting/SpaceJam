## Getting started

SpaceJam is an open-sourced internal package for the visuals of the Mira Collection

## Usage

Add `spacejam` to your dependencies.yaml.
```yaml
dependencies:
  spacejam: ^0.3.0
```

Run pub get in your terminal.
```console
flutter pub get
```
